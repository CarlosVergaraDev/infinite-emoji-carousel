# Infinite Emoji Carousel

A lightweight infinite loop slider built with vanilla JavaScript, HTML5, and CSS3.  
Implements seamless horizontal transitions using CSS transforms and DOM element cloning to simulate continuous motion.

---

## Features

- Infinite loop behavior (no visible jump)
- CSS `transform: translateX()` based transitions
- DOM node cloning for seamless reset
- Autoplay functionality
- No dependencies (pure vanilla JavaScript)

---

## Project Structure

```
infinite-emoji-carousel/
│
├── index.html     # Markup structure
├── styles.css     # Layout and visual styling
└── script.js      # Infinite loop logic
```

## Installation

Clone the repository:

```bash
git clone https://github.com/CarlosVergaraDev/infinite-emoji-carousel.git
```

Open `index.html` in your browser.
